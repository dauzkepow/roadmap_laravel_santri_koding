1_tag selector

memilih elemen berdasarkan nama tag htmlnya

semua element p akan memiliki warna teks biru

2_Class Selector

class digunakan untuk menandai satu atau lebih elemen html

yang memiliki kesamaan

untuk memilih class di CSS gunakan tanda titik (.) didepan nama class

Paragraf ini berwarna merah

Catatan = satu class bisa digunakan oleh banyak elemen

dan satu elemen bisa memiliki beberapa class

3_ID Selector

Digunakan untuk menandai satu elemen unik

Untuk memilih ID di CSS gunakan tanda pagar # didepan nama ID

Ini adalah judul utama

Catatan = ID sebaiknya unik, satu ID hanya digunakan untuk satu elemen di satu halaman

4_Universal Selector

digunakan untuk memilih semua elemen dalam dokumen html

menggunakan *
